Learn how to build the structure of a webpage.

Lesson 1: Basic Structure

HTML uses tags like <html>, <head>, and <body> to structure a page.

<!DOCTYPE html>
<html>
  <head><title>My Page</title></head>
  <body>Hello World!</body>
</html>

Lesson 2: Headings and Paragraphs

Use headings (<h1> - <h6>) and paragraphs (<p>) to organize text.

<h1>Welcome</h1>
<p>This is my first paragraph.</p>

Lesson 3: Links and Images

Use <a> for links and <img> for images.

<a href="https://example.com">Visit Site</a>
<img src="image.jpg" alt="My Image">
